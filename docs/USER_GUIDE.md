# 矿区检测平台 - 使用说明

## 1. 平台简介

矿区检测平台是一个基于遥感影像的矿区扰动检测系统。通过分析 NDVI（归一化植被指数）时间序列数据和裸煤概率图，自动识别可能存在矿业活动的区域。

### 核心功能
- 上传 GeoTIFF 格式的遥感数据
- 运行 KNN-DTW 矿区检测算法
- 在地图上可视化检测结果
- 查看指定位置的 NDVI 时间序列曲线
- 下载检测结果文件
- 管理历史检测任务

## 3. 检测分析

### 3.1 准备数据

需要准备两个 GeoTIFF 文件：

**NDVI 时序文件** (多波段)
- 每个波段代表一年的 NDVI 数据
- 波段顺序应按年份递增排列
- 推荐分辨率: 30m (Landsat)

**裸煤概率文件** (单波段)
- 表示每个像元是裸煤区域的概率
- 值域: 0-1 或 0-100
- 与 NDVI 文件空间范围和分辨率一致

### 3.2 上传文件

1. 在检测页面左侧找到「上传数据」区域
2. 点击「NDVI 时序 GeoTIFF」上传框，选择 NDVI 文件
3. 点击「裸煤概率 GeoTIFF」上传框，选择裸煤文件
4. 设置起始年份（NDVI 第一个波段对应的年份）

### 3.3 运行检测

1. 确认两个文件都已上传（上传框显示文件名）
2. 点击「上传并运行检测」按钮
3. 等待处理完成（通常需要 1-5 分钟，取决于数据大小）
4. 状态栏显示「检测完成」后，地图会自动加载结果图层

### 3.4 查看结果

**地图查看**
- 检测结果以颜色编码显示在地图上
- 可缩放、平移地图查看不同区域
- 右上角图层列表可切换显示的图层

**NDVI 曲线查看**
- 点击地图上任意位置
- 左侧「NDVI 时间序列」面板会显示该像元的:
  - 各年份 NDVI 值折线图
  - 检测结果类别
  - 裸煤概率值

### 4.1 单个文件下载

1. 在检测页面底部找到「下载文件」面板
2. 勾选要下载的文件
3. 点击「下载选中文件」

### 4.2 打包下载

1. 勾选多个文件
2. 点击「打包下载 (ZIP)」
3. 浏览器会下载包含所有选中文件的 ZIP 压缩包

### 4.3 可下载文件说明

| 文件 | 说明 |
|------|------|
| result.tif | 检测结果栅格 (GeoTIFF) |
| result.geojson | 检测结果矢量 (可导入GIS软件) |
| ndvi.tif | 上传的原始NDVI文件 |
| coal.tif | 上传的原始裸煤概率文件 |

## 8. 数据格式要求

### 8.1 NDVI 时序文件

```
格式: GeoTIFF (.tif / .tiff)
波段: 多波段，每波段一年
值域: -1 到 1 (NDVI标准范围)
数据类型: Float32 推荐
坐标系: 任意投影坐标系 (会自动转换)
NoData: 支持 NoData 值
```

### 8.2 裸煤概率文件

```
格式: GeoTIFF (.tif / .tiff)
波段: 单波段
值域: 0-1 (概率值) 或 0-100 (百分比)
空间范围: 必须与NDVI文件一致
分辨率: 必须与NDVI文件一致
```

### 8.3 常见问题

**Q: 两个文件的范围不一致怎么办？**
A: 使用 GDAL 或 QGIS 将两个文件裁剪/重采样到相同范围和分辨率。

**Q: 支持什么坐标系？**
A: 支持任意坐标系，系统会自动转换为 Web Mercator 进行显示。

**Q: 文件太大上传失败？**
A: 系统默认支持 500MB 以内的文件。如需处理更大文件，请联系管理员调整配置。

## 9. 检测结果解读

### 9.1 结果类别

检测结果通常分为以下类别：

| 值 | 含义 | 颜色 |
|----|------|------|
| 0 | 无扰动 | 灰色 |
| 1 | 疑似扰动 | 黄色 |
| 2 | 明显扰动 | 红色 |

### 9.2 NDVI 曲线特征

**正常植被**
- NDVI 值稳定在 0.4-0.8
- 年际变化平缓

**矿区扰动特征**
- NDVI 突然下降
- 恢复缓慢或持续低值
- 与裸煤概率高值区域重合

## 11. 常见问题

**Q: 为什么地图上没有显示结果？**
A: 检查任务状态是否为「已完成」，刷新页面后重试。

**Q: 点击地图没有显示曲线？**
A: 确保点击位置在数据范围内（蓝色边框内）。

**Q: 检测失败怎么办？**
A: 检查数据格式是否符合要求，查看任务详情中的错误信息。
